精确算法 BRANCHING

Posted on 2023-08-17 Edited on 2023-08-28 In 图论 Word count in article: 3.2k Reading time ≈ 3 mins.

basic concepts

branch 有两条规则:

  • reduction rule. 用来简化问题或者停止算法.
  • branching rule. 通过递归地求解更小的问题的实例来解决一个问题.

因此设计一个 branch 算法很大程度上是设计 reduction 和 branching rule.

upper bound analysis

要分析 branching 算法需要一些假设:

  • polynomial time.
  • Simplifying produces simpler problem.

在上面的假设下, 复杂度为 搜索树结点 \(\times\) 一个多项式.

研究 branch 算法时, 问题的输入参数往往不是输入的长度, 而是一个可以描述问题规模的很自然的参数, 比如图的结点数, 称这样的参数为问题的 instance.

设 \(T(n)\) 是 instance 数目为 \(n\) 的一个问题的分支树的叶子节点个数. 给定一个 branching 规则 \(b\), 设 \(r\ge 2\), \(t_i>0,i\in [r]\), 规则 \(b\) 将当前的 instance 分成了 \(r\) 个子问题且每个子问题的 instance 数量至多为 \(n-t_i,i=1,\cdots,r,n\ge \max_{1\le i\le r}t_i\), 那么称向量 \(\mathbf{b}=(t_1,\cdots,t_r)\) 称为 branching vector. 根据树的递归结构, 有 \[ T(n)\le \sum_{i=1}^rT(n-t_i) \] 这是一个线性递推不等式, 考虑用特征方程来解, 等式 \(T^*(n)=\sum_{i=1}^rT^*(n-t_i)\) 的特征方程为 \(x^n-x^{n-t_1}-\cdots-x^{n-t_r}=0\), 而显然有 \(T(n)\le T^*(n)\), \(T^*(n)\) 为特征方程的所有根的 \(n\) 次幂的线性组合, 可以写成 \(T^*(n)=O^*(\alpha^n),\alpha\) 为方程的最大正根, 这就意味着 \(T(n)=O^*(\alpha^n)\).

将对于 branching vector \((t_1,\cdots,t_r)\), 定义 \(\tau(t_1,\cdots,t_r)=\alpha\). 有如下性质:

  • \(\tau(t_1,\cdots,t_r)=\tau(t_{\pi(1)},\cdots,t_{\pi(r)})\), 其中 \(\pi\) 为 \(1,\cdots,r\) 的任意排列.
  • \(\tau(t_1,\cdots)<\tau(t_1',\cdots), t_1>t_1'\).

\(k\)-SAT

令变量集合 \(X=\{x_1,\cdots,x_n\}\land x_i\in\{0,1\},\) 有下面的概念:

  • literal \(x\in X\) 以及 \(\bar{x}\) 被称为 literal, 与 \(X\) 有关的 literal 的集合记作 \(L(X)=\{x_1,\bar{x}_1,\cdots,x_n,\bar{x}_n\}\).
  • clause 一个 clause \(c\) 是若干 literal 的 disjunction \(c=\ell_1\lor \cdots\lor \ell_t,\ell_i\in L(X)\). 要求一个 clause 不含有两个相同的 literal 或者 一个 literal 和它的非. 用集合 \(\{\ell_1,\cdots,\ell_t\}\) 来表示一个 clause.
  • CNF formula 是若干 clause 的conjunction \(F=c_1\land \cdots\land c_m\), 用集合 \(\{c_1,\cdots,c_m\}\) 来表示一个 CNF formula. 若 \(F\) 的每个 clause 都含有至多 \(k\) 个 literal, 则称其为 \(k\)-CNF formula. 如果 CNF formula \(F'\) 满足 \(F'\) 的每个 clause 都是 \(F\) 的每个 clause 的子集, 则称 \(F'\) 是 \(F\) 的 subformula.

称 CNF formula \(F\) 是 satisfiable 的当且仅当存在对 \(F\) 的 literal 的合法赋值使得 \(F=1\). 以此定义 SAT 和 \(k\)-SAT 问题:

  • Satisfiability Problem. 给定一个含有 \(n\) 个变量和 \(m\) 个 clause 的的 CNF formula \(F\), 要求判断是否能找到对变量的合法赋值(对每个变量赋值得到 \(F\) 的一个结果的过程称为 truth assignment)使 \(F=1\).
  • \(k\)-Satisfiability Problem. 在 SAT 的基础上, 要求 \(F\) 是 \(k\)-CNF formula.

\(k\)-SAT 的一个 branch 算法

考虑到对于 \(F\) 的任意 clause \(c=\ell_1\lor \cdots\lor \ell_q\), 当 \(\ell_i=1\) 时便有 \(c=1\), 同时可以得到 \(F'\), 其中 \(F'\) 相比 \(F\) 少了一个变量 . 以这种方式作为分支规则, 可以得到 \(q\) 个 subformula:

  • \(F_1:\ell_1=1\)
  • \(F_2:\ell_1=0,\ell_2=1\)
  • \(F_3:\ell_1=0,\ell_2=0,\ell_3=1\)
  • \(\cdots\)
  • \(F_q:\ell_1=0,\ell_2=0,\cdots,\ell_q=1\)

于是若 \(F_1,\cdots,F_q\) 之中至少要有一个是可满足的, 便可得到 \(F\) 是可满足的.

复杂度分析

由于 \(F\) 有 \(n\) 个变量即 \(n\) 个instance, 于是经过赋值, \(F_i\) 有 \(n-i\) 个 instances, 其 branching vector 为 \((1,\cdots,q)\), 对应的特征方程为 \(x^{n}-x^{n-1}-\cdots-x^{n-q}=0\), 利用等比数列求和化简得 \(x^{q+1}-2x^{q}+1=0\).

\(2\)-SAT 可以在线性时间内解决. \(k\ge 3\) 时 \(k\)-SAT 是 \(\mathsf{NPC}\) 的.

一个简单的 branching

独立集一些简单的结论:

  • 若 \(v\in I\), 则 \(N(v)\cap I=\emptyset\).
  • 若 \(I\) 是 MIS, 则对于 \(v\not\in I\), \(N(v)\) 中至少有一个点在 \(I\) 中即 \(|N(v)\cap I|\ge 1\), 如若不然, \(I\cup\{v\}\) 可称为新的独立集.
  • 对于每一个 \(v\in V\) 和MIS \(I\), 存在一个 \(y\in N[v]\) 使得 \(y\in I\) 但 \(N(y)\cap I=\emptyset\).

第三条怎么和第一条重复了?

下面是递推关系式: \[ \alpha(G)= \begin{cases} 0,& V=\emptyset\\ \displaystyle 1+\max_{y\in N[v],v=\arg\min_{u\in V(G)}\deg (u)}\alpha(G\backslash\ N[y]),&\text{otherwise} \end{cases} \] 对于度数最小的点, 在 \(y\) 和 \(N(v)\) 中选择一个, 然后再把选择的点及其邻域删去, 选择的要求是, 要保证剩下的图的独立集最大.

复杂度为 \(O^*(s^{n/s}),s=\min_{v\in V}\deg(v)+1\), \(s=3\) 时为\(O^*(3^{n/3})\) 是最大值. 这个复杂度分析并不 trivial, 先贴在下面:
