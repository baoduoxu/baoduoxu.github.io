Tree Decomposition

给定图 \(G=(V,E)\), \(G\) 的树分解是一颗树 \(T=(\mathcal{V},F)\), 其中 \(\mathcal{V}\subseteq 2^V\), \(\mathcal{V}\) 中的元素叫做 bag, 满足:

  1. 对任意 \(\{u,v\}\in E\), 则存在 \(V_i\in \mathcal{V}\) 使得 \(u,v\in V_i\)
  2. 对任意 \(v\in V\), 存在 \(V_i\in\mathcal{V}\) 使得 \(v\in V_i\)
  3. 对任意 \(v\in V\), 记 \(\mathcal{V}\) 中包含 \(v\) 的 bag 的集合为 \(\mathcal{V}_v\), 则 \(T[\mathcal{V}_v]\) 是 \(T\) 的一连通分支(一棵子树).

一种更规范的定义是, 树分解是二元组 \((\mathcal{V},T)\), 其中 \(\mathcal{V}=\{V_i:i\in I\}\), \(I\) 为指标集, 树 \(T=(I,F)\). 相当于在树分解后树的结点和bag 之间加了一个双射.

树分解 \((\mathcal{V},T)\) 的宽度为 \(\max_{V_i\in \mathcal{V}}|V_i|-1\), 在 \(G\) 所有的树分解中最小的宽度称为 \(G\) 的树宽, 记作 \(\mathrm{tw}(G)\).

容易验证至少有两个结点的树的树宽为 \(2\), \(K_n\) 的树宽为 \(n-1\).

Path Decomposition

图 \(G\) 的 Path Decomposition 是满足 \(T\) 为链的树分解 \((\mathcal{V},T)\), 因此可以记作 \((V_1,\cdots,V_r)\). 同样可以定义其宽度为 \(\max_{1\le i\le r}|V_i|-1\), 以及所有的 path decomposition 中最小的宽度记作 pathwidth \(\mathrm{pw}(G)\). 显然 \(\mathrm{tw}(G)\le \mathrm{pw}(G)\).

若 \((V_1,\cdots,V_r)\) 满足

  • \(|V_1| =|V_r| =1\);
  • \(V_{i+1}=V_i\cup\{v\}\) 或 \(V_{i+1}=V_i\backslash{\{v\}}\) 成立, 其中 \(v\in V\), \(1\le i\le r-1\). 这也是说进行路径分解后得到的路径中前一个 bag 总是比后一个 bag 多一个结点或者少一个结点.

则称这个路径分解是 好 (nice) 的.

MAX-CUT

对于 \(G\), 其最大割指的是最大化 \((X,V\backslash{X})\) 之间的边数的集合 \(X\). 现在借助路径分解以及动态规划给出一个结果, 记 \(E(V_1,V_2)\) 为 \(G\) 中 \(V_1,V_2\) 之间的边的集合, 即 \[ E(V_1,V_2)=\{\{u,v\}\in E:(u\in V_1\land v\in V_2)\lor(u\in V_2\land v\in V_1)\} \] 给定 \(G\) 的一个宽度至多为 \(k\) 的路径分解, 在线性时间内可以得到好的宽度为 \(k\) 的路径分解 \(P=(V_1,\cdots,V_r)\), 记 \(U_j=\bigcup_{i\le j}V_i\), 则 \(U_{r}=V\).

对于某个 \(V_i\), 令 \((A,B)\) 为 \(V_i\) 的一个划分, 那么 \((A,B)\) 可以确定一些 \(V\) 的划分 \((A',B')\), 满足 \(A\subseteq A',B\subseteq B'\). 记 \[ c_i(A,B)=\max_{A'\supseteq A,B'\supseteq B,(A',B')是V的划分}|E(A',B')\cap E[U_i]| \] 其中 \(E[U_i]\) 为 \(G[U_i]\) 的边集.

那么就需要求 \[ \max_{A,B是 V_r的划分} c_r(A,B) \] 首先考虑边界 \(c_1(A,B)\), 此时设 \(V_1=\{v\}\), 则 \((A,B)\subseteq\{(\{v\},\emptyset),(\emptyset,\{v\})\}\), 此时 \(E[U_i]=\emptyset\), 则对任意 \((A,B)\), \(c_1(A,B)=0\).
