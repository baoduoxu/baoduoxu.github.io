这样 B 成为 A 的子模块(submodule, 后续还是称作子仓库), 那么在主仓库的路径下的 git 操作将无法直接控制子仓库, 可以进入子仓库的路径进行正常的 git 操作, 然后回到主仓库进行正常用的 commit 和 push 即可.
